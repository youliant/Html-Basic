# Html-Basic

Repository ini berisi latihan dasar HTML.

## Isi Project
- `index.html` → halaman utama
- `about.html` → halaman tentang
- `form.html` → halaman form

## Catatan

Kedepannya, akan ditambahkan CSS untuk styling dan JavaScript untuk interaktivitas.
